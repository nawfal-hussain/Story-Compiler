# Story Compiler

## Compiler Construction Project

---

## Current Status

---

## Usage

```bash
python compiler.py
```

---

## Language Keywords

```
STORY, END, SCENE, CHARACTER, SAY, CHOICE, GOTO
```

---

## Token Types

- `STRING` - Text in quotes
- `ID` - Identifiers
- `ARROW` - `->`
- Keywords - Uppercase matches
